## 2. Главная функция

**Записать 3 достижения за день.**

Один экран. Три поля. Написал -- нажал "Done". Всё. Никаких категорий, тегов, приоритетов. Минимализм до предела.

---

## 3. Изюм (чем отличается)

### "Streak Flame" -- визуальная серия

Каждый день записи = день серии. Серия визуализируется как пламя:
- 1-3 дня: маленький огонек (тлеющий уголек)
- 4-7 дней: растущее пламя
- 8-14 дней: яркий костер
- 15-30 дней: горящий факел
- 30+ дней: epic flame с частицами

Пламя живое -- CSS-анимация, оно дышит и колышется. Это единственный декоративный элемент на экране. Пропустил день -- пламя гаснет, начинаешь с нуля.

**Почему это работает**: Bears Gratitude перегружен (медитации, цитаты, категории). Мы -- антитеза. Один ритуал, один визуальный reward. Dopamine loop через streak, как Duolingo, но без guilt-tripping.

### Дополнительные штрихи:
- **Dark-only** дизайн. Journaling -- это вечерний ритуал. Тёмный фон, теплые цвета пламени.
- **"Look back"** -- свайп/скролл вниз показывает прошлые дни как ленту. Без календаря, без графиков. Просто лента записей.
- **Микро-анимации** на каждое действие: ввод текста, сохранение, открытие дня.

## 4. Acceptance Criteria

### MVP (8 критериев)

| # | Критерий | Детали |
|---|----------|--------|
| AC-1 | Главный экран показывает сегодняшний день с 3 пустыми полями ввода | Дата в формате "Saturday, March 29". Три поля с placeholder "Win #1", "Win #2", "Win #3" |
| AC-2 | Пользователь может ввести текст в каждое поле и нажать "Done" | Кнопка "Done" активна только когда заполнено хотя бы 1 поле. После нажатия -- данные сохраняются, поля блокируются |
| AC-3 | Данные сохраняются в localStorage и переживают перезагрузку | Ключ: `threeWins_YYYY-MM-DD`. При открытии страницы -- если сегодня уже записано, показать заполненные поля (readonly) |
| AC-4 | Streak отображается как анимированное пламя | CSS-анимация огня. 5 уровней визуала (1-3, 4-7, 8-14, 15-30, 30+ дней). Число дней серии показано под пламенем |
| AC-5 | Streak считается корректно | Серия = непрерывная последовательность дней с записями, заканчивающаяся сегодня (или вчера, если сегодня ещё не записано). Пропуск 1 дня = reset |
| AC-6 | "Look back" лента показывает прошлые записи | Скролл вниз от главного экрана. Каждый день -- дата + 3 записи. Обратный хронологический порядок. Пустые дни не показываются |
| AC-7 | Можно редактировать запись текущего дня | Кнопка "Edit" рядом с сохраненными записями сегодня. Прошлые дни -- read-only |
| AC-8 | Responsive: выглядит отлично на 375px (iPhone SE) и на 1440px (desktop) | Mobile-first. На десктопе -- контент по центру, max-width 480px |

### Дополнительно (2 критерия)

| # | Критерий | Детали |
|---|----------|--------|
| AC-9 | Анимация сохранения: при нажатии "Done" пламя вспыхивает ярче на 1 секунду | CSS transition / keyframe. Feedback что действие засчитано |
| AC-10 | Пустое состояние (первый запуск): текст "Start your streak. Write your first three wins." вместо пламени | Нет пламени пока нет ни одной записи |

## 5. Чего НЕ делать

| Запрет | Почему |
|--------|--------|
| Регистрация / аккаунты | Нет бэкенда. localStorage only. Без трения |
| Категории / теги / приоритеты | Убивает простоту. Bears сделал эту ошибку |
| Напоминания / push-нотификации | Это веб-приложение, не PWA на первом этапе |
| Статистика / графики / аналитика | Overcomplicated. Пламя -- единственная метрика |
| Экспорт / импорт | Не MVP |
| Цитаты дня / мотивационные фразы | Cringe. Пусть пользователь сам пишет свои wins |
| Light theme | Dark-only на MVP. Вечерний use case |
| Больше 3 полей | Ровно 3. Это constraint by design, не баг |
| Шаринг / social | Wins -- личное. Без social pressure |
| Backend / база данных | Всё в браузере. Zero infrastructure |

## Дизайн-направление

- **Palette**: черный фон (#0A0A0A), теплые тона пламени (от #FF6B35 до #FFD700), белый текст (#F5F5F5)
- **Typography**: системный шрифт (-apple-system), крупный размер (18-24px для полей ввода)
- **Spacing**: щедрые отступы, воздух между элементами
- **Animations**: плавные, 300-500ms, ease-out. Пламя -- бесконечный loop, subtle
- **Vibe**: как будто записываешь у костра. Тёплый, спокойный, не корпоративный
